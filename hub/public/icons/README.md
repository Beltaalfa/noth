# Ícones PWA

Para instalação do Hub como app no celular, coloque aqui:

- **icon-192.png** – 192×192 px
- **icon-512.png** – 512×512 px

Você pode gerar a partir do `logo.svg` com ferramentas como [realfavicongenerator.net](https://realfavicongenerator.net/) ou [pwa-asset-generator](https://www.npmjs.com/package/pwa-asset-generator). Enquanto isso, o manifest usa o `/logo.svg` como ícone.
